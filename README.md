This is the lathe I have been desingning for the past 2-ish months, I was initially inspired by Chris Borge a Youtuber who focuses on creating machines to help with his own personal projects and 3d-prints that are filled with concrete. Now, this is the second iteration design, not becuase I built a first lathe but because the first time I tried to put everything into CAD it all started to look like a toy and I thought I could do better, which I'm glad that I did. Which now leads to the vesion you see above and below this bar of text.

Something I initially struggled with was finding a way to connect all the parts toghether, in my initial design I tried to make each 3d printed part interlock with dove-tails but that soon became much more problematic than was needed. I solved this problem with allowing an open hole from part to part on the inside to connect all parts into this chamber that I hope keeps everything together once the concrete dries. 

For the reason that I built this project, I want to have better manufacturing capabilities at home, and having a machine that allows me to focus on one type of part (being conical and cylindrical parts) I think will allow me to have better projects in the future.

# Assembly
Hi! if you're reading this, your either reading to review, see whats going on, or interested in building this project yourself.


- The first step in creating the lathe is set up, make sure you have all hardware, bearings, belt &amp; pulley's, 3d prints, PCB, concrete, and extrusion material on hand

- Make sure to use heated inserts before this next step and to cover anny holes which may clog with concrete with tape.

- The next step is is to assemble the base, head stock, and upper walls seperately, glue them with each other inside those groups and let it rest (you may want to put the hardware, extrusions, and linear slide in for the base before glueing), after the glue is done drying, you may start off by pouring concrete into the base.

- After concrete has been poured into the base (process can change depending on the part) you can now assemble and put-in the top of the lathe wall and headstock and fill those with conctete as well, if you can find a power tool to vibrate the lathe to allow air bubbles to escape then use that as well too.

-After allowing the concrete to dry, you may add on the tool carriage and screw on the PCB to to the lathe, make sure to wire and test and you should have a functional lathe!

Remember when using the lathe, the part should be spining to where it's coming down onto the tool and not the other way around.

Currently the lathe is run by either two buttons on the back for foward/revevrse but the option for external control was also given, so with an off board micro controller you could be very exact with how you want this thing to spin. (I plan to use an off board MCU when possible)
